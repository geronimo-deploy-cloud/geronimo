# credit-risk

ML model serving API generated by Geronimo.

## Quick Start

```bash
# Install dependencies
uv sync

# Run the API locally
uv run uvicorn credit_risk.api.main:app --reload

# Run tests
uv run pytest
```

## Project Structure

```
credit-risk/
├── geronimo.yaml          # Deployment configuration
├── pyproject.toml         # Python project config
├── Dockerfile             # Container definition
├── azure-pipelines.yaml   # CI/CD pipeline
├── infrastructure/        # Terraform files
├── src/
│   └── credit_risk/
│       ├── api/          # FastAPI application
│       │   ├── main.py
│       │   ├── routes/
│       │   └── models/
│       └── ml/           # Model loading & inference
│           └── predictor.py
├── models/               # Model artifacts
└── tests/
```

## Deployment

```bash
# Generate all deployment artifacts
geronimo generate all

# Deploy infrastructure
cd infrastructure && terraform apply
```
